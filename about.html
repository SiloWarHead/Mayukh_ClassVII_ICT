About This Project

Exploring the intersection of high-performance hardware and modern design.

Project Objective

The goal of this ICT assessment is to demonstrate a clear understanding of web development using HTML5 and CSS3. I chose the theme of "Gaming PC Builds" to showcase how different hardware components work together.

Hardware Categories

In this project, I have categorized hardware into three main areas:

  • Processors: The computational power of the system.
  • Graphics: The visual rendering capabilities.
  • Storage: The speed of data access and file safety.
